Simple command line todo list


## Usage

```
$ todo --help
NAME:
   todo - todo CLI 

USAGE:
   todo [command]

Available Commands:
  add         Adds a task to yout task list
  clean       remove all done tasks
  delete      delete the todo
  do          Marks task a complete
  help        Help about any command
  list        List all of your tasks
  undo        Unmark task a complete

Flags:
  -h, --help   help for todo


```
